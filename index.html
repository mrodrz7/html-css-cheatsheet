HTML Tags

Tag Name Description
<html> HTML Tag Root of HTML Doc - container/wrapper for all HTML elements
<head> Head Tag Container for metadata - typically define title (tab) styles, scripts
<body> Body Tag Defines Doc's body - contains all content (h1-h6, p, img, etc)
<a> Anchor Tag Used to define the beginning and end of hypertext link
<a href> Href Tag Hyperlink Reference - Used to link pages within an html doc

Common CSS Declarations

>
Declaration Description Example
background-color Specifies color to the background of an element background-color: green;
background-image Specifies an image to use for the background of an element background-image: url("https://randomimage.com");
font-family Specifies the font for an element - default is Times New Roman font-family: Arial;
font-weight Sets how thick or thin characters in text should be displayed font-weight: bold;
text-align Aligns text in element either left, right, or center text-align: center;
